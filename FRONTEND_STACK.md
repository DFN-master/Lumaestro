# ?? Stack do Frontend: A Face do Maestro  
---  
## ?? Core Framework  
O projeto utiliza o **Vue.js 3** com a **Composition API**.  
A gerência de estado é feita via **Pinia**, garantindo uma aplicação reativa.  
---  
## ?? Visualização de Dados  
Para o grafo de conhecimento, usamos o **D3.js** e o **D3-Force-3D**.  
Isso permite a simulação física dos nós do **Obsidian** em tempo real.  
---  
## ??? Ferramentas e UI  
O **Vite** é o motor de build que garante o hot-reload veloz.  
Utilizamos **Xterm.js** para a renderização do terminal dos agentes.
